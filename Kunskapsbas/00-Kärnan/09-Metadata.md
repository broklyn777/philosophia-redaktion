# Metadata-standard

## Syfte

Alla dokument i Philosophias Kunskapsbas ska inledas med metadata.

Metadata gör dokumenten konsekventa, sökbara och enkla att återanvända.

Den hjälper både människor, ChatGPT och Codex att förstå dokumentets syfte utan att läsa hela innehållet.

---

# Obligatoriska fält

| Fält              | Beskrivning                       |
| ----------------- | --------------------------------- |
| Status            | Dokumentets aktuella status       |
| Typ               | Dokumenttyp                       |
| Område            | Kunskapsområde                    |
| Ämnen             | Centrala begrepp                  |
| Personer          | Viktiga personer eller författare |
| Källa             | Primär källa eller inspiration    |
| Prioritet         | Låg, Normal eller Hög             |
| Publicerad        | Ja eller Nej                      |
| Skapad            | Datum                             |
| Senast uppdaterad | Datum                             |

# Dokumentstruktur

Efter metadata används normalt följande rubriker när de är relevanta:

1. Frö
2. Kärnidé
3. Centrala begrepp
4. Grundteser
5. Relaterade dokument
6. Möjliga fortsättningar

Alla dokument behöver inte innehålla samtliga rubriker, men metadata är alltid obligatorisk.
